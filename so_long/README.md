# so long
This is 2D game, coded using minilibX.

# Dependency
- You need make, gcc
- You need intel MAC

# How to play

## Rule
- You can move using "WASD" keys.
- You can exit if you get every collectible
- Beware of enemies for bonus!

## Key
- W: move up
- S: move down
- A: move left
- D: move right
- ESC: exit game

## Mandatory
```bash
$ make
$ ./so_long maps/map.ber
```

## Bonus
```bash
$ make bonus
$ ./so_long_bonus maps/bonus.ber
```
